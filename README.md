## 特性

- 生成原生通用组件 icon
- 生成夸平台可用的 iconfont-weapp.css 文件

## 如何安装

```shell

npm  i  -g  iconfont-tools

cd  project/asset/font_hiytajitqeu  // 进入图标文件所在文件夹

iconfont-tools  // 生成小程序专用文件

```
