# iconfont-tools

适用于 iconfont 彩色图标的提取

## 特性

- 生成原生通用组件 icon
- 生成夸平台可用的 icon.css 文件

## 如何安装

```shell

npm i -g iconfont-tools

cd project/asset/font_hiytajitqeu // 进入图标文件所在文件夹

iconfont-tools // 生成小程序专用文件

```

## 目录

```shell

project
│ 
├── asset
│   └── font_hiytajitqeu
│       ├── demo.css
│       ├── demo_index.html
│       ├── iconfont.css
│       ├── iconfont.eot
│       ├── iconfont.js
│       ├── iconfont.svg
│       ├── iconfont.ttf
│       ├── iconfont.woff
│       └── iconfont.woff2
```
